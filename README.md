# ToDoListJava
 make a simple todolist by java
+ Set up JVM or Java SDK
+ Open command prompt(with Windows)
+ cd "the directory has todolist_java.jar"
+ Enter "java -jar todolist_java.jar"
(Hope you enjoy it!)
